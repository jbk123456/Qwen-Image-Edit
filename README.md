# Qwen Image Edit ComfyUI Demo

## Installation

* Download Qwen2.5-VL-7B-Instruct-mmproj-BF16.gguf and qwen2.5-vl-7b-instruct-q4_k_m.gguf to models/clip
* Download Qwen_Image_Edit-Q4_K_M.gguf to models/unet
* Download Qwen_Image-VAE.safetensors to models/vae
* Start comfyui using: podman-compose up
* Open the UI at http://127.0.0.1:8188/
